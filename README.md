<h1>💳 Credit Card Customer & Transaction Analysis</h1>

<p>
This repository contains a Power BI dashboard project focused on analyzing credit card transactions, customer behavior, and segmentation insights. The dashboard helps understand spending patterns, customer engagement, and potential churn indicators.
</p>

<h2>📊 Overview</h2>

<h3>1. Credit Card Transaction Analysis</h3>
<p>
This report provides insights into overall transaction performance.
</p>
<ul>
  <li>Total Transaction Amount and Total Transaction Count</li>
  <li>Transaction comparison across different card categories (Blue, Silver, Gold, Platinum)</li>
  <li>Analysis of transaction trends based on customer activity</li>
  <li>Identification of high-performing customer segments</li>
</ul>

<h3>2. Customer Segmentation Report</h3>
<p>
This report focuses on understanding customer profiles and behavior.
</p>
<ul>
  <li>Customer distribution by Age Group, Education, and Income Category</li>
  <li>Analysis of spending patterns across different segments</li>
  <li>Comparison of transaction activity among various customer groups</li>
  <li>Identification of high-value customer segments</li>
</ul>
